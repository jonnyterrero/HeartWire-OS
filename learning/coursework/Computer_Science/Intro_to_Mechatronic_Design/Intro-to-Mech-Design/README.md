# Intro to Mechatronic Design

Course repository for **Arduino programming**, **circuit analysis**, and **embedded/computer architecture** work.

## Repo structure

- **`labs/`**: individual lab assignments and experiments
- **`final-project/`**: final project iterations and related code
- **`robotic-arm/`**: robotic arm sketches, FSMs, kinematics, and supporting assets
- **`class-materials/`**: handouts, slides, datasheets, references (stored via Git LFS)

## How to use

- **Arduino sketches**: open any `*.ino` in the Arduino IDE.
- **Support files**: some folders include wiring notes (`.md`) or exported assets used for the build.

## Notes

- Large binaries (PDF/PPTX/DOCX/XLSX/images/zips/exe) are tracked with **Git LFS** so cloning and pushing stays reliable.
- If `final-project/diagram and layout for final project .pdf` is missing locally, re-copy that file from your OneDrive `School/Intro to Mechatronic design` folder (OneDrive can time out on large or cloud-stub files during batch sync).
