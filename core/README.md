# core/ — Foundational Documentation

This directory contains the foundational documents that define HeartWire OS.

## Files

- **`mission.md`** — Why this workspace exists and what it represents.
- **`systems-architecture.md`** — Explanation of the folder structure and how all components interconnect.
- **`learning-roadmap.md`** — Long-term plan for self-study across all disciplines.
- **`onboarding.md`** — Instructions for collaborators and AI agents on how to navigate and use HeartWire OS.
- **`ai-agent-rules.md`** — Rules all AI assistants (ChatGPT, Claude, Cursor, Perplexity, etc.) must follow when operating inside this repo.

## Purpose

To create a cohesive, extensible system that organizes:

- academic learning
- engineering projects
- self-study
- automation and AI agents
- health-app and biomedical tooling
- long-term professional growth
